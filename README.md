# YouTube Video Downloader (Python)

This project provides a simple way to download YouTube videos using Python and the `pytubefix` library.

## ⚠️ Important Note

The standard `pytube` package can sometimes break due to YouTube updates.  
To avoid issues, this project uses **pytubefix**, a maintained version of pytube.  
It is recommended to install it inside a **virtual environment** so it doesn’t interfere with your system Python setup.

---

## 📦 Requirements

- Python 3.7+
- pip (Python package manager)
